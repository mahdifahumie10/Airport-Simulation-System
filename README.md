# Airport-Simulation-System

# DESCRIPTION
- Designed an aiport simulation system using Java on Apache Netbeans to demonstrate concurrency concepts such as threads, deadlock avoidance, wait/notify, etc.

# FEATURES
- CLI - acts as an ATC (Air Traffic Control) system that prints messages of what is happening at the airport
- Makes use of concurrency concepts to simulate the management of limited resources such as 2 runways, 1 refuel truck etc. among several planes
- Simulates an emergency/congestion scenario

# HOW TO RUN
- Click the green Code button and Download ZIP
- Navigate to the extraced AirportJava folder
- Open the folder using Netbeans, and run. The simulated scenario and the ATC messages can be seen.
